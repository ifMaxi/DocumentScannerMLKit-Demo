# Document Scanner ML Kit Demo

DEMO application to put ML Kit into practice. In this case, it is a simple document scanner where you can capture, save, and access the generated PDF file. For simplicity, everything was written in the MainActivity file.

> [!NOTE]
> Requires Android 12 or higher to use the app.

### Tech Stack
- Kotlin
- Jetpack compose
- Ksp
- Hilt
- Document Scanner from ML Kit
- Coil
